SenseMap

User Guide

SenseMap is available as a Chrome extension. Make sure you have the latest version (0.5.4). Install and click on the icon at the top-right corner of your browser to open the extension. In short, SenseMap will capture, visualize your browsing activities, and allow you to curate your collection to suit your needs.

Browser

  • You can select a passage of text and highlight it using the context menu. Click on the highlight to remove or add a note.
  • A snapshot of a webpage is used to help you recognize it. You can replace that snapshot with a more representative picture in the webpage by right-clicking on the picture you want and select Set as Page Image.

Collection View

Automatically collects and displays your browsing activities
Node
  • Each node represents a webpage with its logo and title.
  • Keyword search pages are detected to show with the search icon and keyword.
  • Highlights and nodes are displayed below but limit to the maximum of 2. Click on Show All hovering menu to see all of them.
  • You can go back to a webpage by clicking on the corresponding node.
  • Click on the highlight within a node to go back to that specific highlight.
  • You can highlight, minimize, curate and remove a node using the hovering menu.
  • Node's background color: cyan for active page, yellow for favorite page, and red for brushing between two views.
  • Dashed border indicates unvisited pages
  • Curated nodes have a little tick at the top right cornder.
  • Click on a minimized node to maximize it.
    • Link
  • Link shows where a page comes from. Note: please wait for the visiting page completely loaded before clicking on any links to make sure the link detection work properly. Sometimes, still wrong though!
    • Menu
  • Click on the three-bars icon to open the toolbar. You can create a new map, save, or load an existing one.
  • You can pause and resume collecting information.
  • There are several ways to pan the map
    • Click on an empty space and drag.
    • Click on any place while holding Shift key and drag.
    • Mouse wheel for vertical pan.
    • Track pad.
  • You can zoom the map using plus/minus button at the bottom right corner of the window.

    • Curation View

    Curate your collected information
    Node
  • Nodes are added from the collection view by Curate button in the hovering menu.
  • Nodes are always shown with pictures.
  • You can remove a node using the hovering menu.
    • Link
  • Links from collection view will be copied to the curation view when nodes are added.
  • You can add custom link by holding the left mouse button until the cursor changes to the arrow, then dragging it and dropping onto the targeting node.
  • Custom links are shown with blue color to distinguish with system links.
  • You can remove a link using the hovering menu.
    • Unsaved data will be gone if you close SenseMap windows.
