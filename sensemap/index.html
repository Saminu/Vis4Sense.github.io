SenseMap

User Guide

SenseMap is available as a Chrome extension. Install and click on the icon at the top-right corner of your browser to open the extension. In short, SenseMap will capture, visualize your browsing activities, and allow you to curate your collection to suit you best.

Browsing Features

  • You can select a passage of text and highlight it using the context menu. Click on the highlight to remove or add a note.
  • A snapshot of a webpage is used to help you recognize it. You can replace that snapshot with a more representative picture in the webpage by right-clicking on the picture you want and select Set as Page Image.

Map Interaction

Map
  • Click on the three-bars icon to open the toolbar. You can create a new map, save, or load an existing one.
  • There are several ways to pan the map
    • Click on an empty space and drag.
    • Click on any place while holding shift and drag.
    • Mouse wheel for vertical pan.
    • Track pad.
    • Node
  • Each node represents a webpage with its logo and title.
  • You can go back to a webpage by clicking on the corresponding node.
  • Click on the highlight within a node to go back to that specific highlight.
  • You can highlight, minimize, and remove a node using the hovering menu.
  • Click on a minimized node to maximize it.
  • Click on a removed node to put it back.
    • Link
  • Link shows where a page comes from.
  • You can add custom link by moving the mouse close to the edge of a node (note the mouse cursor changes to the right arrow) then dragding it and dropping on the target.
  • You can hide or completely remove a link using the hovering menu.
  • Click on a hidden link to show it.
    • Manual Mode
  • Available in the toolbar.
  • Once use that mode, the layout won't be updated when nodes and links are modified. However, new nodes can't be added as well. So, it may only be used when you've done your collecting.
    • A note for evaluation: please save your map when finish.
